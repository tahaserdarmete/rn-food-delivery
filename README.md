# 🍔 Food Delivery App

A modern and user-friendly mobile food ordering application. This project was developed using **React Native** and **Expo** to provide a user experience suitable for real-world scenarios.

---

## 📝 Notes

This project currently uses **Mock Data**. The infrastructure for API integration is ready via Redux Toolkit during the development phase.

---

## 🚀 Features

* Restaurant Listing
* Cart Management
* Map Integration
* Modern UI
* Navigation

---

## 🛠 Technologies Used

* Framework
* State Management
* Styling
* Navigation
* Maps
* Icons
